raapchikftp
===========

single click ftp tool


Installation:
 - Python 2.7
 - Paramiko
 - Py2Exe

run command "python setup.py py2exe"

It will create 2 folders dist and build...

In dist folder right click [open as administrator] "install_smftp.exe"
